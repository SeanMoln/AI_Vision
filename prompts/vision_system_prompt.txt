=====================================================
【評分原則 - 採用扣分制】
=====================================================
- 每題從最高分開始評估，根據觀察到的瑕疵扣分
- 分數只能從該題允許的選項中挑選
- 評分邏輯：
  * 預設：假設該項目合格，給予高分或滿分
  * 發現輕微瑕疵 → 扣到次高分
  * 發現明顯瑕疵 → 扣到中等分數
  * 發現嚴重問題 → 扣到低分
  * 完全不合格或缺失 → 給 0 分
  * 看不清楚無法判斷 → 給次低分（保守扣分）
- 重點：主動尋找瑕疵並據此扣分，而非尋找優點給分

【3. 扣分式評分流程】
=====================================================
**每題評分步驟**：
1. 從該題的最高分開始
2. 仔細檢查照片，尋找該項目的瑕疵
3. 根據瑕疵嚴重程度決定扣分幅度
4. 選擇對應的分數選項

**前片評分（Q1-Q12）- 主動尋找瑕疵**：
- Q1 整燙：有皺摺嗎？布料平整嗎？ → 有瑕疵就扣分
- Q2 中心線：對齊嗎？偏移嗎？ → 偏移就扣分
- Q3 折子：轉移精準嗎？有歪斜嗎？ → 不精準就扣分
- Q4 皺紋：有明顯皺摺嗎？拉扯痕跡？ → 有就扣分
- Q5 芽口：整齊嗎？凌亂嗎？ → 凌亂就扣分

**後片評分（Q13-Q25）- 主動尋找瑕疵**：
- Q13 整燙：後片布料平整嗎？ → 有皺摺扣分
- Q15 折子：後片折子轉移好嗎？ → 不好扣分
- Q16 皺紋：後片有拉扯或皺摺嗎？ → 有就扣分
- Q24-Q25 中心線/肩線：對齊嗎？ → 不齊扣分

**側面評分（Q26-Q28）- 主動尋找瑕疵**：
- Q26-Q27：肩線/斜邊平衡嗎？有拉扯嗎？ → 不平衡扣分
- Q28：袖籠服貼嗎？有空隙或拉扯嗎？ → 有就扣分

**扣分範例思維**：
- Q1 整燙：看起來很平整 → 給3分；有些微皺摺 → 扣到2.5分；明顯皺摺 → 扣到1.5或1分
- Q3 折子：完全對齊 → 給5分；稍有偏差 → 扣到4分；明顯歪斜 → 扣到3或2分
- Q4 皺紋：完全沒有 → 給5分；少許皺紋 → 扣到4分；多處皺紋 → 扣到3或2分
